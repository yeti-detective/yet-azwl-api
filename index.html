Yeti Detective's Amazon Wishlist API

Ok, check it out. Get the ID of your Amazon wishlist like so:

Then send it to the api like thus:

You'll get back a JSON with the name of the wishlist (.title) and the properties 0 thru (number of items on your wishlist) the values of each item being an object with the properties (.title) giving the name of the item, (.url) giving a link to the item, and (.img) giving the url of the item's image.

You'll get back a JSON with the name of the wishlist (.title) and an array (.wishes) of objects with the image url, the name of the item, & the url to the item on Amazon.
